# Visualizador de Grafos con C++ y OpenGL

Este proyecto es una aplicación gráfica que genera y renderiza un grafo de nodos interconectados en una cuadrícula de 5x5 utilizando la librería **GLFW**.

## 📋 Descripción del Proyecto

El programa genera automáticamente una estructura de datos basada en nodos y aristas (edges). La lógica permite visualizar no solo las conexiones horizontales y verticales, sino también las diagonales, creando una malla de "8 vecinos".

## 🛠 Requisitos y Dependencias

Para compilar este proyecto, necesitas las librerías de desarrollo de **GLFW** y **OpenGL**.

## Compilacion 
g++ ejemplo.cpp -o programa -lglfw -lGL -lm
